# GH Workspace Manager V4.1 Release Notes

## Version 4.1 - Dashboard and Workspace Health

This release introduces the first working application features beyond the project scaffold.

## Added

- Dashboard page
- Workspace path setting
- Required workspace folder health check
- Missing folder detection
- Create missing folders action
- Basic Git availability check
- Basic Git repository status page
- Persistent settings file
